# Peako Light Prototypes

Logic prototypes for mjpeakeART's [Time Pieces](http://mjpeake.com/time-pieces).

## Development

`$ yarn start`

## Tests

`$ yarn test`

## Build

`$ yarn build`

## Boilerplate

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
